Copulae
=======

A `copula <https://en.wikipedia.org/wiki/Copula_(probability_theory)>`_  is a multivariate probability distribution for
which the marginal probability distribution of each variable is uniform.

Presently, only elliptical and `Archimedean copulas`__ have been implemented.

__ https://en.wikipedia.org/wiki/Copula_(probability_theory)#Archimedean_copulas
